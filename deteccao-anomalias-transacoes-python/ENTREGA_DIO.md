## Descricao da tarefa

```text
Projeto desenvolvido para o desafio "Deteccao de Anomalias em Transacoes em Python". Criei uma base simulada de transacoes bancarias e implementei um script em Python para identificar operacoes suspeitas com base em regras estatisticas e criterios de negocio, como valor muito alto, horario incomum, transacao internacional e saque elevado. O repositorio contem a base CSV, o codigo Python, o arquivo de saida com as anomalias detectadas e a documentacao explicando metodologia, insights, limitacoes e melhorias futuras.
```

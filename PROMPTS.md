# Prompts usados no caderno tematico

Este arquivo registra os prompts testados durante o desafio **Treinando uma IA de Aprendizagem: Explore o Poder do NotebookLM**.

## 1. Resumo inicial

```text
Explique o que e IA generativa.
```

Problema encontrado: resposta ampla e generica.

Versao melhorada:

```text
Com base apenas nas fontes adicionadas, explique o que e IA generativa em linguagem simples. Depois, compare IA generativa com machine learning tradicional e inclua 3 exemplos praticos de uso.
```

## 2. Comparacao de conceitos

```text
Qual a diferenca entre IA generativa e LLM?
```

Problema encontrado: resposta correta, mas pouco estruturada.

Versao melhorada:

```text
Crie uma tabela comparando IA generativa, LLMs e engenharia de prompts. Para cada item, apresente: definicao, funcao principal, exemplo de uso e risco comum.
```

## 3. Revisao ativa

```text
Crie perguntas sobre o conteudo.
```

Problema encontrado: perguntas simples demais.

Versao melhorada:

```text
Crie 10 perguntas de revisao ativa sobre IA generativa, LLMs e prompts. Misture perguntas conceituais, perguntas de aplicacao pratica e perguntas que obriguem a comparar ideias. Inclua o gabarito comentado.
```

## 4. Glossario

```text
Faca um glossario.
```

Problema encontrado: glossario curto e sem exemplos.

Versao melhorada:

```text
Monte um glossario com os principais termos do caderno. Para cada termo, traga uma definicao simples, um exemplo pratico e um erro comum de interpretacao.
```

## 5. Plano de estudos

```text
Crie um plano de estudos sobre IA generativa.
```

Versao melhorada:

```text
Crie um plano de estudo de 7 dias sobre IA Generativa, LLMs e Engenharia de Prompts. Para cada dia, inclua: objetivo, topicos a estudar, atividade pratica, pergunta de revisao e fonte recomendada.
```

## 6. Prompt reutilizavel para novos temas

```text
Estou estudando [TEMA]. Com base nas fontes deste caderno, atue como um tutor de aprendizagem ativa. Primeiro, crie um resumo estruturado. Depois, gere um glossario, 10 perguntas de revisao e 5 prompts que eu possa usar para continuar estudando.
```

# 02 - Base de Conhecimento

## Arquivo principal

A base esta no arquivo:

```text
data/base_conhecimento.json
```

## Estrutura

Cada tema contem:

- titulo
- palavras-chave
- resposta principal
- passos sugeridos
- proxima acao

## Temas cadastrados

| Tema | Objetivo |
| --- | --- |
| orcamento | Ajudar a organizar renda, despesas e saldo mensal. |
| reserva_emergencia | Explicar como criar uma reserva para imprevistos. |
| dividas | Orientar a priorizacao e renegociacao segura. |
| cartao_credito | Apoiar uso consciente do cartao e da fatura. |
| pix_seguranca | Reforcar boas praticas para evitar golpes. |
| metas | Ajudar a transformar objetivos em planejamento mensal. |

## Estrategia

A base foi mantida pequena para facilitar teste, manutencao e explicabilidade. Em uma evolucao futura, ela poderia ser substituida por documentos, FAQs, banco vetorial ou integracao com um LLM.
